** Root cause and ownership

=kitty-latest= publishes four outputs: =out=, =terminfo=, =shell-integration=,
and =kitten=.  =guix/systems/guixpc.scm= installs only the default =out= output.
The profile therefore lacks the separate
=share/terminfo/x/xterm-kitty= entry even though =TERMINFO_DIRS= already names
the correct System profile =share/terminfo= directory.

Adding the old package-private =lib/kitty/terminfo= directory to the environment
makes the probe pass, but that is diagnostic evidence, not the fix.  The narrow
declarative correction is to retain =kitty-latest= and add its =terminfo= output
to the System recovery package list:

#+begin_src scheme
(list kitty-latest
      (list kitty-latest "terminfo"))
#+end_src

** Problem and evidence

Truthful Kitty SSH sessions export =TERM=xterm-kitty=, but GuixPC installs only
Kitty's =out= output.  =infocmp xterm-kitty= fails while =xterm-256color= and the
Emacs server are healthy.  Manually redefining =TERM= hides the ownership bug.

** Accepted scope

- [ ] Add the separate =terminfo= output beside =kitty-latest= in the GuixPC
  System recovery package list.
- [ ] Add a regression probe for both =xterm-kitty= and =xterm-256color=.
- [ ] Prove noninteractive SSH remains byte-clean.
- [ ] Preserve one Fastfetch greeting in interactive local and SSH Fish.
- [ ] Prove the Emacs server RPC and a paired real =emacsclient -t= session.
- [ ] Build, checkpoint, activate, and cold-accept the System generation.

** Explicit non-goals

- Replacing Fish or changing its greeter.
- Setting =TERM=xterm-256color= globally or overriding the connecting terminal.
- Adding package-private terminfo paths to shell startup files.
- Changing Kitty, Emacs, SSH daemon, or EXWM behavior beyond the missing output.
- Beginning performance tuning, GC, theme, or Lisp-dialect work.

** Ownership and files likely affected

- =guix/systems/guixpc.scm=: add =(list kitty-latest "terminfo")= while
  retaining the existing default output.
- Existing focused validation scripts/tests: add the smallest terminfo and SSH
  contract regression that fits their current ownership.
- This document and the agenda roadmap: record build/live/reboot evidence.
