* Interpretation boundary

=favorable= means the fresh cached snapshot contained none of the reader's
selected failing indicators and no selected unknown that would force the
classification to =unknown=.  It does not mean:

- every possible ATA SMART attribute was inspected;
- a SMART self-test was run or passed;
- the drive is guaranteed healthy now or in the future;
- wear or remaining lifetime was established; or
- performance, filesystem integrity, discard behavior, or backup health was
  measured.

The =success-or-never= self-test label is intentionally ambiguous: UDisks does
not prove from that cached value alone that a self-test ever ran.
